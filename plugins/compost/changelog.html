Compost Changelog

To install Compost, unpack the entire compost directory into the /plugins/ folder on your server. After this, enter the Plant Site Admin and enable it on the Plugins tab. Done!

  • 2012-07-15: Version 1.2

    • Changed methods:
      • PostController->actionMain() - Compatibility issue for Plant version 1.0
      • PostController->actionManage() - Compatibility issue for Plant version 1.0
      • PostController->actionSearch() - Compatibility issue for Plant version 1.0
    • Misc. Added:
      • Cache_Lite - Caching class for Filebrowser

    2011-09-15: Version 1.1

    • Changed methods:
      • FilebrowserController->actionDelete() - Now uses File instead of ContentFile
      • PostModel->edit() - Now resets date_posted when status changes from draft to published
    • Added methods:
      • FilebrowserController->actionGet() - Method to dynamically retrieve file info via AJAX
    • Misc. Changed:
      • config.inc.php - Added new directives REMOTE_MEDIA_ROOT, FILEBROWSER_ENCODING_RATE and FILEBROWSER_ENCODING_DELAY
      • post-edit.tpl - Fixed bug that wouldn't display video or images, added post thumb field
    • Misc. Added:
      • filebrowser-ajax.tpl - For retrieving file info via AJAX

  • 2011-02-17: Version 1.0 (hooray)

    • Changed methods:
      • FilebrowserController->actionDelete() - Now uses File instead of ContentFile
      • FilebrowserController->actionMain() - Fully rewritten using uploadify and better file indexing
      • FilebrowserController->findFiles() - Replaced by FilebrowserController->scan()
      • FilebrowserController->setProperties() - Added cache options
      • FilebrowserController->uploadFile() - Replaced by FilebrowserController->uploadFiles()
      • PostController->actionEdit() - Small edits to take post type into account
      • PostController->populateForm() - Edits to take post type into account

      • PostModel->edit() - Edits to take post type into account
      • PostModel->getContent() - Edits to take post type into account
      • PostModel->getExcerpt() - Edits to take post type into account
    • Added methods:
      • FilebrowserController->actionGet() - Method to dynamically retrieve file info via AJAX
      • FilebrowserController->process() - Processes files in different ways depending on file type
      • FilebrowserController->scan() - Scans directory and indexes files using caching
      • FilebrowserController->uploadFiles() - Uploads and processes a list of incoming files
      • PostController->actionAdd() - Override method to deal with new posttype URL token
      • PostModel->getParameters() - Retrieve type-specific post parameters
      • PostModel->getType() - Retrieve current post type
      • PostModel->getTypes() - Retrieve types statically
      • PostModel->getVideo() - Find and return video in post content
    • Misc. Changed:
      • admin-post.css - Extra styles for different post types
      • admin-post-nav.tpl - Added links for different post types
      • ContentFile.class.php - Removed (functionality now in Plant File class)

      • config.inc.php - Added new configuration directives for the Filebrowser
      • filebrowser.css - New filebrowser styles
      • filebrowser.js - New JS frontend for filebrowser
      • filebrowser.tpl - Fully rewritten filebrowser frontend
      • feed-posts.tpl - Added generic author email address
      • post.tpl - Small edits to handle optional titles and post types
      • PostModel.class.php - Added type column
      • post-edit.tpl - Edits to take post type into account
      • post-manage.tpl - Edits to take post type into account
      • post-single.tpl - Minor changes to formatting
      • postcategory-manage.tpl - Rewritten to use a proper table for display
      • quicktags.js - Added support for other video types, support for different post types
    • Misc. Added:
      • filebrowser-ajax.tpl - For retrieving file info via AJAX
      • uploadify.js - Uploadify Flash library for filebrowser multiple file uploading support
      • uploadify.swf - Uploadify Flash library for filebrowser multiple file uploading support
      • css/images/ - Images directory with images for uploadify (upload-new.png, upload-cancel.png, delete-icon.png) and file types (audio.png, file.png, video.png, loadingAnimation.gif)
