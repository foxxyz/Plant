Plant Installation Guide

Welcome To The Greenhouse!

Thanks for trying out Plant! This has been one of my biggest projects to date, and also the one of which I am most proud of. For me the main reason to make this framework was to make building a site fun again, regardless of scale. I hope Plant will give you as much joy as it did me.

System Requirements

To make the most flexible system possible, I've embraced some of the more recent technologies available. If your host doesn't provide these, urge them to upgrade!

  • Apache 1.3 or higher
  • PHP 5.2 or higher
  • MySQL 4.0 or higher
  • The mod_rewrite module installed for Apache

Starting Out

  1. Unzip the package into its own directory
  2. Modify config.local.inc.php in the /app/config/ directory and fill in the details of your database connection.
  3. Upload all the files to your web server!
  4. Set write permissions on the following directories so Plant can access data:
    • /app/controllers/
    • /app/templates/
    • /core/cache/
    • /content/
  5. Run /install/ on your site. Follow the steps, and note the generated password at the end.
  6. Delete the /install/ directory on your server.

That's it! The installation script will send you to the Site Admin where you'll be able to change your password and start constructing your site.
